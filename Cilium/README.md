# Cilium

Operational runbooks and manifests for deploying Cilium across AKS, EKS, and GKE environments.

## Contents
- `How To's/`: Troubleshooting notes, clustermesh guidance, and deployment steps for advanced networking scenarios.
- `Manifests/`: Egress gateway policies, gateway API annotations, FRR configs, and sample workloads tailored to multiple clouds.

## Usage
Pair the how-to documents with the manifests referenced inside them to keep Cilium experiments reproducible.
