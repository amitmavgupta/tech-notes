# SDN ACI OpenStack

Resources for integrating Cisco ACI with OpenStack.

## Files
- `aci_drivers_mechanisms_comparisons.md`: Overview of the available ML2 drivers.
- `aci_opflex_plugin.md`: Architecture note for the OpFlex plugin.
- `aci_opflex_architecture.png`, `gbp_apic_mapping.png`, `ml2_standard_neutron_plugin_mapping.png`: Reference diagrams.

Use these when evaluating or presenting ACI-backed OpenStack deployments.
