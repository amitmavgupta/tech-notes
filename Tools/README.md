# Tools

Installation notes, configuration snippets, and cheat sheets for frequently used developer and operations tools.

## Highlights
- Installation guides for AWS CLI, Azure CLI, kubectl, helm, docker, stern, tmux, and more.
- Git/GitHub workflows (`git_commands.md`, `github_auth_vm.md`).
- Networking utilities such as `tcpdump.md`, `bgpd_conf.md`, and `zebra_conf.md`.

Reference these markdown files whenever you are setting up a new workstation or need a quick command reminder.
