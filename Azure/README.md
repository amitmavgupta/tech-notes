# Azure

Azure-focused networking, compute, and CLI references plus supporting diagrams.

- Automation runbooks: `auto_upgrade_azure_cli.md`, `create_acr.md`, `enable_nested_virt.md`.
- Networking guidance: `azure_load_balancer_dns_name_annotation.md`, `load_balancer_permissions.md`, `troubleshooting_azure_route_server.md`.
