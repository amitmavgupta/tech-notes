# Elastic Kubernetes Service-Anywhere

Runbooks and manifests for EKS Anywhere (EKS-A) installations.

## Contents
- `How To's/`: Operational guides covering private clusters, upgrades, connector management, and add-ons.
- `Manifests/`: EKS-A defaults plus AWS EKS Connector resources for registering clusters with the AWS console.

Review the relevant how-to before applying any manifest to keep installations reproducible.
